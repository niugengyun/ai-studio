## 📥 下载安装

1. 打开 Releases：**[点这里下载](https://github.com/niugengyun/ai-studio/releases)**  
2. 在页面里选一个版本，展开 **Assets**，下载对应系统的压缩包或镜像：
   - **Windows**：找到带 `windows` 的 `.zip` → 解压 → 双击 **`ai-studio.exe`**
   - **macOS**：找到带 `macos` 的 `.dmg` → 打开 → 把应用拖到 **应用程序** → 从启动台打开  
3. 优先选 **最新** 那条 Release。

## ❓ 打不开、闪退、白屏怎么办

### Windows

1. 🔧 [安装 WebView2 运行时（Evergreen）](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/) — 装好再启动一次程序  
2. 📋 仍不行：打开日志文件，把报错相关内容发给作者  
   - 路径：`%LOCALAPPDATA%\AIStudio\launcher.log`

### macOS

1. 📋 打开日志：`~/Library/Logs/AIStudio/launcher.log`，把报错相关内容发给作者
